# Исследование рынка венчурных инвестиций в стартапы

## Введение
Проект посвящён исследовательскому анализу венчурных инвестиций в стартапы с целью выявить закономерности финансирования, динамику развития компаний и определить наиболее перспективные направления для инвестиций.  
Данные взяты из базы стартапов **Crunchbase** и охватывают период до 2015 года.  

---

## Используемые библиотеки
pandas, numpy, seaborn, matplotlib, phik

## Шаг 1. Знакомство с данными: загрузка и предобработка

Основной датасет находится в архиве cb_investments.zip, внутри которого — файл cb_investments.csv.

Описание данных

Основной датасет (cb_investments.csv):

name — название компании

homepage_url — сайт компании

category_list — категории, в которых работает компания

market — основной рынок (отрасль)

funding_total_usd — общий объём инвестиций, $

status — текущий статус компании

country_code, state_code, region, city — географические данные

funding_rounds, participants — количество раундов и участников

founded_at, founded_month, founded_quarter, founded_year — даты основания

first_funding_at, mid_funding_at, last_funding_at — даты финансирования

Финансовые показатели по типам инвестиций:

seed, venture, equity_crowdfunding, undisclosed,
convertible_note, debt_financing, angel, grant,
private_equity, post_ipo_equity, post_ipo_debt,
secondary_market, product_crowdfunding, round_A–round_H.

Дополнительный датасет (cb_returns.csv):

year — год возврата

Суммы возвратов по типам инвестиций (seed, venture, angel, grant и др.) в миллионах долларов.

Проект является учебным, все базы данных принадлежат разработчику курса.

## Цель проекта

Провести исследовательский анализ венчурных инвестиций в стартапы, чтобы выявить закономерности финансирования, динамику развития компаний и определить перспективные направления для инвестиций.

## Задачи

Провести предобработку данных.

Разделить компании по срокам существования и количеству раундов финансирования.

Классифицировать сегменты рынка на массовые, средние и нишевые.

Определить типичные и аномальные значения объёмов инвестиций.

Сравнить популярность и объёмы различных типов финансирования.

Проанализировать динамику раундов и объёмов инвестиций по годам.

Рассчитать долю возврата средств по видам инвестирования.

Сформулировать рекомендации по инвестиционным стратегиям.
